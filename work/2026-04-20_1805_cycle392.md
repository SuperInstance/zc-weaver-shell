## 1. **plato-torch** → **plato-ensign** — **CONNECTED**
- **Connection:** Direct export pipeline.
- **Evidence:** `plato-torch/src/rooms/deadband_room.py` contains:
  ```python
  from plato_ensign.export import export_room_lora
  ```
- **Data Flow:** Room training logs → LoRA adapter via `export_room_lora()`.
- **Integration Point:** `export_room_lora()` called after room training cycles.

---

## 2. **plato-torch** → **fleet-simulator** — **NOT CONNECTED**
- **Gap:** No imports or shared configuration found.
- **Evidence:** `plato-torch/` has no reference to `fleet-simulator` in source or requirements.
- **Potential Channel:** Both use training data (tiles/simulations) but no direct pipe.
- **P1 Opportunity:** Could connect via a shared tile format or relay service.

---

## 3. **holodeck-rust** → **plato-torch** — **PARTIALLY CONNECTED**
- **Connection:** Sentiment-aware NPCs generate interaction tiles.
- **Evidence:** `holodeck-rust/src/npc/sentiment.rs` defines `Tile` struct with fields matching `plato-torch` tile schema.
- **Data Flow:** NPC interactions → JSON tiles → unknown ingestion point.
- **Integration Gap:** No explicit import or API call from holodeck to plato-torch. Tiles likely written to disk or sent via unlinked relay.

---

## 4. **fleet-simulator** → **holodeck-rust** — **NOT CONNECTED**
- **Gap:** Separate repos, no shared modules.
- **Evidence:** `fleet-simulator/src/` contains own agent simulation logic; no Rust dependencies.
- **P1 Opportunity:** Could feed simulated agent behaviors into holodeck as NPC templates.

---

## 5. **plato-ensign** → **fleet-simulator** — **NOT CONNECTED**
- **Gap:** Ensign exports LoRA adapters; simulator does not consume them.
- **Evidence:** `plato-ensign/` only exports to file system; no reference to simulator.
- **Potential Channel:** Simulator could load ensign adapters to vary agent behaviors.

---

## 6. **Ghost Tiles** — **ISOLATED COMPONENT**
- **Location:** `plato-torch/src/rooms/ghost_injector.py`
- **Status:** Reads from `memory/ghost_tiles/` but not wired into holodeck or simulator.
- **Integration Gap:** Ghost tiles are P0 knowledge (dead agents' lessons) but not injected into live training loops.
