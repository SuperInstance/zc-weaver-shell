## 1. Verified Connections (Based on Code)

### **plato-torch → fleet-simulator**
- **Connection:** Strong, direct dependency.
- **Evidence:** `plato-torch/src/rooms/fleet_simulator_room.py` imports from `fleet_simulator` (`from fleet_simulator import FleetSimulator`).
- **Purpose:** `FleetSimulatorRoom` uses the simulator to generate training data (tiles) from simulated fleet interactions.
- **Integration Point:** The room calls `simulator.step()` and processes the resulting `events` into training tiles.

### **plato-torch → plato-ensign**
- **Connection:** Strong, direct dependency.
- **Evidence:** `plato-torch/src/export/ensign_exporter.py` imports `plato_ensign`.
- **Purpose:** Exports a trained room's experience (accumulated tiles) into a LoRA adapter via `plato-ensign`.
- **Integration Point:** `EnsignExporter.save_lora()` calls into `plato_ensign`'s API to create and save the adapter.

### **plato-torch → holodeck-rust**
- **Connection:** **Indirect / Planned**. No direct import found in main code.
- **Evidence:** `plato-torch/docs/ROADMAP.md` lists "Holodeck Integration" as a future item. No Rust bindings or direct calls in current Python source.
- **Gap:** No live data flow. Likely intended for future sentiment-aware NPC interactions to generate training tiles.

### **fleet-simulator → holodeck-rust**
- **Connection:** **None found**.
- **Evidence:** `fleet-simulator/src/` contains no references to `holodeck`. It simulates abstract fleet agents, not MUD NPCs.
- **Gap:** Separate domains: fleet agent simulation vs. interactive narrative environment.

### **fleet-simulator → plato-ensign**
- **Connection:** **None found**.
- **Evidence:** No import or usage. The simulator generates event data, not trained models.
- **Gap:** `plato-ensign` operates on room tile history, not raw simulator events.

### **holodeck-rust → plato-ensign**
- **Connection:** **None found**.
- **Evidence:** `holodeck-rust/` is a Rust MUD server. No Python or `plato-ensign` references.
- **Gap:** No export path for NPC interactions to become ensigns.
