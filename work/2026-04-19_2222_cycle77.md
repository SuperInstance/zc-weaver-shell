## 2. Integration Gaps (What's Not Connected)

### **Gap 1: plato-ensign → holodeck-rust**  
- **Issue:** No direct feedback loop from trained ensigns back to holodeck NPCs  
- **Impact:** NPCs cannot improve from exported room experience  
- **Priority:** P1 (safe channel to explore)  
- **Action:** Wire GhostInjector into holodeck (from backlog)

### **Gap 2: DeadbandRoom → plato-relay**  
- **Issue:** Deadband Protocol implementation not connected to relay system  
- **Impact:** Protocol stays theoretical rather than operational  
- **Priority:** P1  
- **Action:** Connect DeadbandRoom to plato-relay (from backlog)

### **Gap 3: fleet-simulator → constraint-theory-core → plato-torch**  
- **Issue:** No documented pipeline for geometric constraint learning  
- **Impact:** Simulation geometry not feeding back to training rooms  
- **Priority:** P2 (optimization gap)  
- **Action:** Test end-to-end pipeline with constraint data

### **Gap 4: Ghost tiles → active training**  
- **Issue:** Ghost tiles (dead agents' lessons) exist as P0 knowledge but not actively integrated  
- **Impact:** Historical lessons not proactively preventing repeated failures  
- **Priority:** P0 (negative space mapping)  
- **Action:** Document integration points for ghost tile injection

### **Gap 5: Mirror Play → current architecture**  
- **Issue:** Two PLATO systems iterating screen-to-screen concept not implemented  
- **Impact:** Missing cross-system training amplification  
- **Priority:** P2  
- **Action:** Research implementation in current codebase
