## 1. Current Verified Connections

### **plato-torch ↔ plato-ensign**  
- **Connection:** Direct and functional.  
- **Mechanism:** `plato-torch` exports training room experience (interaction tiles) via `plato-ensign`'s `export_room()` function.  
- **Evidence:**  
  - `plato-ensign/src/lib.rs` defines `export_room(room_id: &str, adapter_path: &str) -> Result<LoraAdapter>`.  
  - `plato-torch/src/rooms/deadband_room.rs` includes `use plato_ensign::export_room` and calls it in `save_training_adapter()`.  
- **Data Flow:** Room tiles → LoRA adapter file (`.safetensors`).  
- **Status:** **Connected and operational.**

### **plato-torch ↔ fleet-simulator**  
- **Connection:** Indirect via shared tile format.  
- **Mechanism:** `fleet-simulator` generates synthetic training data (JSON tiles) that can be ingested by `plato-torch` rooms.  
- **Evidence:**  
  - `fleet-simulator/src/tile_generator.rs` produces `Tile { id, agent_id, content, timestamp }`.  
  - `plato-torch/src/tile_store.rs` has `ingest_tile(tile: Tile)` — compatible format.  
- **Data Flow:** Simulated agent interactions → JSON tiles → tile store.  
- **Status:** **Compatible but not wired.** No direct API call or pipeline observed.

### **holodeck-rust ↔ plato-torch**  
- **Connection:** None observed.  
- **Mechanism:** Potential via sentiment-aware NPC interactions becoming training tiles.  
- **Evidence:**  
  - `holodeck-rust/src/npc/sentiment.rs` tracks NPC emotional state.  
  - `plato-torch` has no import of `holodeck_rust` nor hooks for NPC events.  
- **Data Flow:** Not connected.  
- **Status:** **Disconnected.**

### **holodeck-rust ↔ fleet-simulator**  
- **Connection:** None observed.  
- **Mechanism:** Could simulate NPC populations.  
- **Evidence:** No shared modules or imports.  
- **Status:** **Disconnected.**

### **plato-ensign ↔ fleet-simulator**  
- **Connection:** None observed.  
- **Mechanism:** Could export adapters from simulated training.  
- **Evidence:** No calls from simulator to ensign.  
- **Status:** **Disconnected.**
