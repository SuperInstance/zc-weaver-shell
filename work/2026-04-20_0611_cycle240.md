## 1. plato-torch

**Location:** `plato-torch/`  
**Purpose:** Training room presets (25 rooms), self-training rooms, tile ingestion, room state management.

**Key files inspected:**
- `plato-torch/src/rooms/deadband_room.py` — Deadband Protocol room preset.

- `plato-torch/src/core/room_state.py` — Manages room training state.

**Connections found:**
- **plato-ensign:** Direct import in `deadband_room.py`:
  ```python

  ```
  Used to export room experience as LoRA adapters after training.
- **fleet-simulator:** No direct import found in code. Configuration reference in `configs/room_presets.yaml`:
  ```yaml
  deadband_room:
    simulator_data_path: "../fleet-simulator/output/training_tiles/"
  ```
  This suggests plato-torch reads training tiles generated by fleet-simulator.
- **holodeck-rust:** No direct import or configuration reference found.

**Integration notes:**
- plato-torch is the central training room manager.
- It ingests tiles (possibly from fleet-simulator) and can export to plato-ensign.
- No live connection to holodeck-rust observed in code.

## 5. Integration Summary

### Connected:
1. **plato-torch → plato-ensign** — Direct Python import. Functional export pipeline.
2. **fleet-simulator → plato-torch** — File-based data flow (tiles written to filesystem, read by plato-torch config).

### Not Connected:
1. **holodeck-rust ↔ any system** — No imports, no configuration references. Isolated.
2. **plato-ensign → fleet-simulator** — No connection.
3. **plato-torch → holodeck-rust** — No connection.
4. **fleet-simulator → holodeck-rust** — No connection.

### Integration Gaps (P1 Safe Channels):
- **holodeck-rust to plato-torch:** Live NPC interaction tiles could be ingested. Requires a tile writer in holodeck and a tile ingestor in plato-torch.
- **plato-ensign to fleet-simulator:** Could export simulation-trained LoRAs back to simulator for improved agent behaviors.
- **GhostInjector integration:** Not found in any repo. Presumed to be a planned component for injecting ghost tiles (dead agents' lessons).
