## 2. Identified Integration Gaps (What's Not Connected)

### **Gap 1: holodeck-rust → plato-torch tile pipeline**  
- **Missing:** No tile exporter in holodeck-rust  
- **Impact:** Sentiment NPC data not feeding training rooms  
- **Action:** Wire GhostInjector (tile formatter) into holodeck output

### **Gap 2: plato-ensign → fleet-simulator feedback loop**  
- **Missing:** Trained ensigns not deployed back to simulator  
- **Impact:** No closed-loop training (simulation → training → improved agents → simulation)  
- **Action:** Add ensign loading to simulator agent config

### **Gap 3: DeadbandRoom → plato-relay**  
- **Missing:** DeadbandRoom preset not connected to relay system  
- **Impact:** Room isolation — no cross-room communication  
- **Action:** Connect via plato-relay message bus (from backlog)

### **Gap 4: flux-runtime → cudaclaw integration**  
- **Missing:** Self-assembling bytecode VM not hosted on GPU runtime  
- **Impact:** VM execution limited to CPU  
- **Action:** Port flux-runtime to cudaclaw CUDA kernels

### **Gap 5: constraint-theory-core → holodeck-rust**  
- **Missing:** Geometric snapping not used for NPC movement  
- **Impact:** NPC navigation simplistic  
- **Action:** Integrate constraint solver for pathfinding
