**Location:** `plato-torch/`  
**Purpose:** Training room presets and self‑training rooms.

**Connections found:**

  `presets/DeadbandRoom.py` imports `from plato_ensign import export_lora`.  
  The `export()` method calls `export_lora()` to create a LoRA adapter from room experience.

  `trainer/self_train.py` imports `from fleet_simulator import FleetSim`.  
  Uses `FleetSim` to generate synthetic training data for rooms.
- **To holodeck‑rust:**

- **To cudaclaw / flux‑runtime:**  
  No imports or references in the inspected Python files.

**Integration gaps:**
- GhostInjector is defined but not wired to holodeck‑rust.
- No connection to plato‑relay (messaging layer) observed.
- No explicit use of constraint‑theory‑core.
