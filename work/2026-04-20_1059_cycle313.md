## 1. plato-torch (Training Rooms)

**Location:** `plato-torch/`  
**Purpose:** 25 training room presets, self-training rooms.  
**Key Files:**
- `presets/deadband_room.py` тАФ Deadband Protocol training room.
- `presets/__init__.py` тАФ Exports presets.
- `core/room.py` тАФ Base Room class.
- `core/tile.py` тАФ Tile data structure.

**Connections:**

  - `plato-torch/core/room.py` has method `export_to_ensign()` that calls `ensign_exporter.export_room(room)`.  
  - **Status:** Connected (direct import, functional export path).

- **fleet-simulator:** Provides training data.  
  - `plato-torch/data/simulator_loader.py` imports `from fleet_simulator.scenarios import ScenarioLoader`.  
  - Used in `DeadbandRoom` to load scenario data for training tiles.  
  - **Status:** Connected (data dependency).

- **holodeck-rust:** Not directly imported. No Rust bindings in Python code.  
  - **Status:** Not connected (no imports, no shared data structures).

**Integration Gaps:**
- No direct connection to `holodeck-rust`.  
- No connection to `cudaclaw` or `flux-runtime` (GPU/VM runtime).  
- `DeadbandRoom` preset does not import `constraint-theory-core`.
