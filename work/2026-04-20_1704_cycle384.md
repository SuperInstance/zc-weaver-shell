## 1. plato-torch → fleet-simulator
**Connection:** **Yes** — Direct import.  
**Evidence:** `plato-torch/src/rooms/deadband_room.py` imports `from fleet_simulator.simulator import FleetSimulator`.  
**Purpose:** The DeadbandRoom uses FleetSimulator to generate training data (simulated agent interactions) for tile creation.  
**Data Flow:** Simulator runs → produces interaction logs → room ingests logs → creates training tiles.  
**Integration Gap:** None observed; the import is functional and used in `step()` method.

## 2. plato-torch → holodeck-rust
**Connection:** **No** — No direct import found.  
**Evidence:** Searched `plato-torch/src/` for "holodeck". No imports or references.  
**Potential Indirect Link:** Both may share tile data format, but no code-level integration seen.  
**Integration Gap:** Direct wiring missing. Could connect holodeck NPC sentiment data to room training.

## 3. plato-torch → plato-ensign
**Connection:** **Yes** — Export pathway.  
**Evidence:** `plato-torch/src/export/ensign_exporter.py` exists. It reads room training checkpoints and converts to LoRA adapters via `plato-ensign` library.  
**Purpose:** After room trains on tiles, ensign_exporter packages experience into a specialist model (ensign).  
**Data Flow:** Room training loop → checkpoint saved → exporter runs → LoRA adapter generated.  
**Integration Gap:** Exporter is separate script; not automatically triggered by room. Manual step.

## 4. fleet-simulator → holodeck-rust
**Connection:** **No** — No import found.  
**Evidence:** `fleet-simulator/src/lib.rs` has no "holodeck" references.  
**Potential:** Both are Rust projects; could share simulation events via API, but no observed connection.  
**Integration Gap:** Holodeck's sentiment-aware NPCs not used in fleet simulations.

## 5. fleet-simulator → plato-ensign
**Connection:** **No** — No direct link.  
**Evidence:** Simulator outputs logs, not ensign adapters. plato-ensign works from plato-torch checkpoints.  
**Integration Gap:** Simulator data must pass through plato-torch room first to become ensign.
