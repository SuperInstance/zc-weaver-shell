## 1. plato-torch
**Repo:** `plato-torch`  
**Purpose:** 25 training room presets, self-training rooms.  
**Key Files:** `rooms/deadband_room.py`, `rooms/__init__.py`, `trainer/room_trainer.py`, `configs/room_configs.yaml`.

### Verified Connections:
- **To plato-ensign:**  
  - `plato-torch/rooms/base_room.py` imports `from plato_ensign.export import export_lora`.  
  - `plato-torch/trainer/room_trainer.py` calls `export_lora()` after training loop.  
  - Direct dependency: `plato-ensign` listed in `pyproject.toml` under `dependencies`.

- **To fleet-simulator:**  
  - No direct imports found in code.  
  - Indirect: `plato-torch/configs/room_configs.yaml` contains `simulation_seed` and `agent_count` parameters that *could* be consumed by fleet-simulator, but no explicit wiring.

- **To holodeck-rust:**  
  - No Rust imports or FFI bindings.  
  - `plato-torch/rooms/holodeck_bridge.py` exists but is a stub—contains `TODO: connect to holodeck-rust via gRPC`.

### Integration Gaps:
- `holodeck_bridge.py` is unimplemented.  
- No shared session state or real-time interaction with holodeck-rust.  
- Simulation parameters are static YAML, not live from fleet-simulator.

## 2. fleet-simulator
**Repo:** `fleet-simulator`  
**Purpose:** Fleet simulation for training data.  
**Key Files:** `src/simulator/core.rs`, `src/agents/mod.rs`, `config/sim_config.toml`.

### Verified Connections:
- **To plato-torch:**  
  - `fleet-simulator/src/output/plato_exporter.rs` writes `.jsonl` logs in format matching `plato-torch/data/tiles/`.  
  - Exported files are manually copied; no automated ingestion pipeline.

- **To holodeck-rust:**  
  - `fleet-simulator/src/environment/holodeck_env.rs` defines `HolodeckEnvironment` struct.  
  - Contains `connect(addr: &str)` method—intended to link to holodeck-rust server, but commented: `// TODO: establish WebSocket`.

- **To plato-ensign:**  
  - No references found.

### Integration Gaps:
- Plato exporter is one-way (simulator → static files).  
- Holodeck environment is defined but not connected.  
- No simulation feedback loop into plato-torch training.
