## 1. plato-torch → fleet-simulator  
**Connection:** **Yes** — Direct import.  
**Evidence:**  
- `plato-torch/src/rooms/deadband_room.py` imports `from fleet_simulator import FleetSimulator`.  
- `DeadbandRoom` instantiates `FleetSimulator` as `self.simulator`.  
- Simulator provides `step()` method called within room's training loop.  
- Shared data: `self.simulator.get_state()` returns observation dict used by room.  

**Integration Gap:**  
- Simulator state format not explicitly documented in plato-torch.  
- Room assumes specific keys (`agent_positions`, `obstacles`, `channels`) but no validation.  

---

## 2. plato-torch → plato-ensign  
**Connection:** **Yes** — Optional export pipeline.  
**Evidence:**  
- `plato-torch/src/exporter/ensign_exporter.py` exists.  
- `DeadbandRoom` has method `export_to_ensign()` that calls `EnsignExporter.save_lora()`.  
- Exporter reads room’s `self.interaction_tiles` and converts to LoRA adapter.  

**Integration Gap:**  
- Export is manual (`room.export_to_ensign()`), not automatic after training.  
- No configuration to auto‑export after N tiles.  

---

## 3. plato-torch → holodeck-rust  
**Connection:** **No** — No direct imports or calls found.  
**Evidence:**  
- Searched `plato-torch/src/` for “holodeck” — zero matches.  
- `holodeck-rust` is a separate MUD server; no Python bindings visible in plato-torch.  
- Potential link: both consume/produce “sentiment” data, but no code connection.  

**Integration Gap:**  
- **Major gap:** Ghost tiles from dead agents (P0 knowledge) could flow from plato-torch to holodeck’s NPCs, but no pipeline exists.  
- No shared session ID or tile format between systems.

## 4. fleet-simulator → holodeck-rust  
**Connection:** **No** — Independent components.  
**Evidence:**  
- `fleet-simulator` is pure Python simulation of fleet dynamics.  
- `holodeck-rust` is a Rust MUD server with WebSocket API.  
- No imports, no shared network layer.  

**Integration Gap:**  
- Simulator could provide environment state to holodeck for immersive training, but no hook.
