**Connection:** **Strong**

## 2. plato-torch → holodeck-rust

## 4. fleet-simulator → holodeck-rust

- No shared data structures or API calls.

**Integration Gap:** **No bridge between simulation and sentiment‑aware MUD.** These are parallel systems.
