## 1. plato-torch → fleet-simulator
**Connection:** ✅ **Connected via data pipeline**  
- **plato-torch/rooms/deadband_room.py** imports `from fleet_simulator.simulator import FleetSimulator` (line 9).  
- **plato-torch/rooms/deadband_room.py** creates a `FleetSimulator` instance in `__init__` (line 29).

## 2. plato-torch → plato-ensign
